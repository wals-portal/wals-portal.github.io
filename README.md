# Мій WALS-портал для Media Station X (MSX) — v2

Компактний, легальний і гарний портал: іконки офіційних сервісів
(через легкий favicon-сервіс, ~2-5 КБ на іконку — швидко вантажиться навіть
на старих ТВ), кольорові плитки, годинник і дата зверху екрана.

## Що змінилось порівняно з v1

- Кожна плитка тепер має справжню іконку сервісу замість плейсхолдера.
- У кожної плитки свій фірмовий колір фону (YouTube — червоний, Spotify —
  зелений, MEGOGO — жовтий і т.д.).
- Зверху екрана з'явився живий годинник і дата (`extension`) — суто
  косметична штука, майже нічого не важить.
- Іконки беруться через `https://www.google.com/s2/favicons?...` —
  це публічний сервіс Google для отримання маленької іконки будь-якого
  сайту за доменом. Легкий, швидкий, не потребує хостингу власних картинок.

## Структура

```
wals-portal/
├── start.json
└── data/
    ├── menu.json     # бокове меню (категорії)
    ├── apps.json      # обране / загальні застосунки
    ├── movies.json      # кіно та ТБ (легальні сервіси)
    ├── music.json         # музика
    └── tools.json           # твій хомлаб (qBittorrent, Home Assistant, MikroTik)
```

## Встановлення

1. Заміни `wals-portal.github.io/msx` у `start.json` та `menu.json` на
   свою реальну адресу (наприклад підпапку в `voipoddsns.github.io`).
2. Онови IP-адреси локальних сервісів (`qBittorrent`, `Home Assistant`,
   `MikroTik`) на свої реальні.
3. Перевір посилання на Lampa (`http://lampa.mx/`) — постав
   свою актуальну адресу, якщо вона змінилась.
4. Залий у GitHub Pages.
5. На ТВ: Media Station X → Settings → Start Parameter → Setup → введи
   адресу без `https://`, наприклад:
   `ТВІЙ_АДРЕС/msx/start.json`

## Як додати нову плитку

```json
{
    "id": "unique_id",
    "title": "Назва",
    "image": "https://www.google.com/s2/favicons?sz=128&domain=example.com",
    "color": "msx-blue",
    "action": "link:window:https://example.com"
}
```

Заміни `example.com` на домен потрібного сервісу — іконка підтягнеться
автоматично, нічого завантажувати вручну не треба.

Доступні кольори: `msx-red`, `msx-blue`, `msx-green`, `msx-yellow`,
`msx-black`, `msx-gray`, `msx-glass`, `msx-white`
(повний список: https://msx.benzac.de/wiki/index.php?title=Colors)

## Чому так, а не важкими картинками

Замість того щоб тягнути великі банери/скріншоти для кожного застосунку
(як роблять деякі сторонні портали), тут використані маленькі іконки-фавікони
— це в рази легше і швидше вантажиться на слабких Smart TV, а виглядає
акуратно завдяки кольоровим плиткам.
